# weatherapp
•	Built a fully functional real-time weather application using the OpenWeatherMap API.
•	Features include city search, GPS-based location detection, live temperature, humidity, wind speed, visibility, and pressure data.
•	Implemented 5-day weather forecast, sunrise/sunset times, and Celsius/Fahrenheit toggle.
•	Designed a clean, responsive dark-theme UI with smooth animations and quick-city shortcuts
